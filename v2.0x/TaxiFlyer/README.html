Alaska Taxi Flyer (Mission-X v2.06+)

Written by Snagar

Mission in brief:
You need to fly students from Chefornak to Bethel in Alaska.
There are two mission files. They are the same in content, except one has custom weather settings to make the flight more chalenging.

Mission Settings:
Secenry:
* [must] Download sceneries: ERAlaska Destinations (both of them).
You can use the maps they add to their scenery.
* [must] OpenScenery X

Rendering settings:
* [uncheck] runway follow terrain.

Enjoy...


Installation Instructions:

Custom Scenery to Download:

  • The mission was built with ERAlaska Destinations in mind - This is a freeware scenery.
  • Open Scenery X a must have scenery package.

    Unpack all sceneries files under "Custom Scenery".

    For Windows and Linux OSes

  • Unzip/Unpack the mission file into a temporary directory.
  • Copy the "Custom Scenery" and "Resources" directory to your {X-Plane Installation Dir}, if prompted to overwrite the directories, click yes!

    When the copying processes will finish, all relevant directories and files would be placed in the correct locations.

    OSX installation

  • Unzip/Unpack the mission file into a temporary directory.
  • Copy the content of "Custom Scenery" (under mission directory) to your XP10 "Custom Directory" (do not overwrite the directory itself since OSX will replace the whole directory).
    Do the same to the "mission" file. Copy the mission XML file located under "Resources/plugins/missionx/data" to the "X-Plane/Resources/plugins/missionx/data" directory.

    If under OSX you were prompted to overwrite the directories, click NO!!!

    You are ready to start the Mission. Load the mission, and go over the briefer instructions.
